# Toy Robot Simulator

This is a ruby practice for a code problem called Toy Robot Simulator.

## Prerequisites

Ruby, Rspec

## Test 

There are 3 test cases. 

Test a)
```
PLACE 0,0,NORTH
MOVE
REPORT
```

Test b)
```
PLACE 0,0,NORTH
LEFT
REPORT
```

Test c)
```
PLACE 1,2,EAST
MOVE
MOVE
LEFT
MOVE
REPORT
```

Run the following command to do the above 3 tests.
```
rspec spec
```
